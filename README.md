# AI Shopping Chatbot

An AI-powered shopping chatbot built using **Next.js** and the **OpenAI API**.  
Users can interact with the chatbot to view products, ask questions, add items to a cart, and complete a basic checkout flow.

---

## Features
- Product listing via chat
- Price and budget-based queries
- Add products to cart
- Simple checkout confirmation
- OpenAI-powered conversational responses
- Deployed on Vercel

---

## Tech Stack
- Next.js (App Router)
- TypeScript
- OpenAI API
- Local JSON data for products
- Vercel (Deployment)
